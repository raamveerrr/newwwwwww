🍕 Simple Food Ordering App

A mobile-responsive React web app for students to pre-order food and skip the queue!

🏠

Home Page

Welcome message with "Order Now" button

📱

Menu Page

Browse food items with category filters and add to cart

🛒

Cart Page

Review selected items, adjust quantities, see total

💳

Payment Page

Mock payment with multiple payment methods

✅

Success Page

Payment confirmation with unique order token

📱

Mobile Responsive

Fully optimized for mobile devices using Flexbox/Grid

🚀 Key Features Implemented:

  • React with Vite: Modern, fast development setup
  • React Router: Smooth navigation between pages
  • Context API: Cart and order state management
  • Mobile-First Design: Responsive CSS with Flexbox/Grid
  • Mock Payment: Simulated payment flow with token generation
  • Clean UI: Beautiful gradients and glass morphism effects
  • Static Data: No backend required - all data is static
  • Accessibility: Proper focus states and responsive design

📁 File Structure:

src/
├── SimpleApp.jsx (Main app with routing)
├── SimpleHome.jsx (Home page component)
├── SimpleMenu.jsx (Menu page with food items)
├── SimpleCart.jsx (Cart page with item management)
├── SimplePayment.jsx (Mock payment page)
├── SimpleSuccess.jsx (Success page with token)
├── SimpleCartContext.jsx (Cart state management)
├── SimpleOrderContext.jsx (Order state management)
└── CSS files for styling

🎯 How to Use:

  1. Start from the Home page with welcome message
  2. Click "Order Now" to browse the menu
  3. Add items to cart using category filters
  4. Review cart and proceed to payment
  5. Simulate payment success
  6. Get your unique order token on success page
🚀 Ready to Run!

Run npm run dev to start the development server
